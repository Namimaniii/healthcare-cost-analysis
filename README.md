# Healthcare Cost Analysis & Data Story

## Overview
This project analyzes healthcare insurance data to identify key factors that influence medical costs, including age, BMI, and smoking status.

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- Data Visualization

## Key Findings
- Smoking is the largest driver of high healthcare costs
- BMI and age also significantly impact charges

## Business Impact
These insights can help insurance providers improve pricing strategies and help healthcare organizations identify high-risk patients earlier.
